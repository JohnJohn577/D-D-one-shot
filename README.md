# D-D-one-shot

A small D&D-inspired one-shot prototype.

## Simple Combat Mode

This repo now includes a short SRD-style one-enemy combat mode with:

- Player HP and AC
- Enemy HP and AC
- Attack rolls vs AC (`d20 + attack bonus`)
- Damage rolls on hit
- Critical hit on natural 20
- Maximum of 5 rounds to keep combat short

### Run

```bash
python3 combat_mode.py
```
